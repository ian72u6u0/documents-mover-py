# PDF Organizer

A simple Python script that automatically finds all PDF files in your **Downloads** folder and moves them to your **Desktop/Documents** folder.

## How to Run It

1. Open your terminal or PowerShell.
2. Run the script:
   ```powershell
   python main.py
   ```

## Built With
* **Pathlib** & **OS**: To find folders on your computer.
* **Shutil**: To safely move the files.
